# 文档导航

本文档目录用于说明仓库文档的阅读顺序，帮助你先找到入口，再进入更细的主题页。

## 入口页

- [`../README.md`](../README.md)，项目快速入口
- [`API.md`](API.md)，API 与 CLI 总索引
- [`structure.md`](structure.md)，仓库结构参考
- [`env.md`](env.md)，开发环境与构建上下文

建议按下面顺序阅读：

1. [`../README.md`](../README.md)，快速了解项目定位、当前能力和基本用法。
2. [`API.md`](API.md)，查看当前集中维护的库接口与 CLI 说明。
3. [`structure.md`](structure.md)，查看仓库结构与目录职责参考。
4. [`env.md`](env.md)，查看 Rust 工具链、构建、测试与开发环境约定。
5. [`API/library/index.md`](API/library/index.md)，进入库接口分组导航。
6. [`API/cli/index.md`](API/cli/index.md)，进入命令行文档分组导航。

当前这些页面主要承担导航和角色定义，不等同于完整参考手册。更细的实现细节和示例以各分组页面、源码和测试为准。

如果你已经知道要看哪一块，可以直接进入 [`API/library/index.md`](API/library/index.md) 或 [`API/cli/index.md`](API/cli/index.md)。
