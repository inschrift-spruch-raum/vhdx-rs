- Evidence:
  - 旧测试只断言 `is_err()`：

- Risk:
  - 同语义多份测试并存且断言强度不一致，会导致未来语义调整时“部分测试过、部分测试碎裂”的维护噪音。
- Disposition: accepted
- Rationale:
  - 当前不做范围外重构/删测；保持现有测试稳定性优先。

2) [LOW] implicit invariant spread across dynamic read/write paths
- File/Line: src/file.rs:309-327, 355-375, 569-587, 1516-1545
- Evidence:
  - `chunk_ratio` 与 BAT payload index 交错规则在 read/write/create 三处重复计算。

  - 未来若修改 BAT 交错策略，需同步多点更新，漏改将引入行为分叉。
- Disposition: accepted
- Rationale:
  - 当前实现逻辑一致且已有回归覆盖（Task 11 + Task 16 gates）；本任务禁止范围外重构。
